Nessa página você pode acessar ou saber um pouco mais sobre os projetos que tenho em andamento.
Minerva 🔗

Implementado em Python, oferece uma interface por linha de comando para renovar os empréstimos na biblioteca da UFRJ.

O uso é muito simples: 

/unix$ minerva [...]
\windows> python -m minerva [...]
WordArTeX 🔗

Um pacote para LaTeX que permite inserir o clássico layout de títulos WordArt, presente no Word 97-2003, em documentos pdf.

Brafman-Assembly 🔗

Um pequeno compilador conforme a linguagem das notas de aula de Sistemas Digitais, ministradas pelo prof. Brafman.
