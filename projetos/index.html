Minerva 🔗

Implementado em Python, oferece uma interface por linha de comando para renovar os empréstimos na biblioteca da UFRJ.

Instalação:

$ git clone repo
$ cd minerva
$ python3 setup.py install

A instalação pode demandar privilégios de admnistrador. Isso requer o uso do comando sudo em sistemas Unix ou executar o prompt de comando como administrador no Windows.

Uso:

  • Linux, OSx:
    • $ minerva [...]
  • Windows:
    • > python -m minerva [...]
As opções são:
  • Renovar os livros:
    minerva [login] [senha]
  • Renovar e armazenar as credenciais:
    minerva -c [login] [senha]
  • Renovar os livros das credenciais armazenadas:
    minerva -r
WordArTeX 🔗

Um pacote para LaTeX que permite inserir em documentos pdf o clássico layout de títulos WordArt, presente no Word 97-2003.

Uso:

\usepackage{wordartex}
